## Getting Started

### Option 1: Using Python (Recommended)

```bash
cd circuit-seeker
python3 -m http.server 8000
```

Then open your browser and navigate to: `http://localhost:8000`

### Option 2: Using Node.js

```bash
npm install -g http-server
cd circuit-seeker
http-server -p 8000
```

### Option 3: Using PHP

```bash
cd circuit-seeker
php -S localhost:8000
```
